LS - mostra o que tem onde estamos
LS -A - mostra os arquivos ocultos ou LS -Force \p. P.Shell 
CD nome do local - abre o diretório que colocamos
PWD - mostra o caminho real que vocę está
MKDIR ‘nome da pasta’ - cria a pasta
GIT INIT . - vira uma pasta especial para versionar ela. Torna ela 
um repositório. 

GIT ADD - Adiciona meus arquivos para commit no repo local.

GIT COMMIT -m  - Faz um save das alteraçőes nos arquivos do repo local

GIT PUSH - Salva minhas alteraçőes no repo remoto.
